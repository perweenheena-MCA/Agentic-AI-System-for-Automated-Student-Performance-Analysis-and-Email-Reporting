# AI-Driven Feedback Summarization and Automated Reporting System Using Agentic AI Workflow Automation

## Overview
The AI-Driven Feedback Summarization and Automated Reporting System is an intelligent web application that automates the process of collecting, analyzing, summarizing, and reporting user feedback. The system uses Google Gemini AI for feedback summarization, Langflow for AI workflow automation, Google Sheets for cloud-based data storage, and Brevo SMTP for automated email notifications.

## Features
- User-friendly feedback submission interface
- AI-powered feedback summarization using Google Gemini
- Automated workflow using Langflow
- Secure storage of feedback in Google Sheets
- Automatic report generation
- Email notification using Brevo SMTP
- Interactive dashboard for administrators
- Real-time feedback analysis

## Technologies Used
- React.js
- Node.js
- Langflow
- Google Gemini AI API
- Google Sheets API
- Brevo SMTP
- Git & GitHub
- Visual Studio Code

## System Workflow
1. User submits feedback.
2. Langflow receives the feedback.
3. Google Gemini AI summarizes the feedback.
4. The summary is stored in Google Sheets.
5. A structured report is generated.
6. The report is emailed via Brevo SMTP.
7. The administrator reviews the report on the dashboard.
